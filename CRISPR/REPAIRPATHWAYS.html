REPAIR PATHWAYS

  • NON-HOMOLOGOUS END JOINING (NHEJ)

    In NHEJ, we leave the cut in DNA to be healed by the enzymes. This process is error-prone and often uses mutations and results in the gene simply turned off. By turning off a problematic gene we can solve some of the issues it causes.

  • HOMOLOGY DIRECTED REPAIR(HDR)

    We can use a DNA template to help DNA double helix break heal. This is less error-prone. We can use this to add new DNA to a sequence if we want to. This can fix mutations and cure some genetic diseases.

 

 

 

These are the two main repair processes that are currently being further investigated by scientists.
